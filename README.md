# 📊 Sales & Revenue Analysis Dashboard

## 📌 Project Overview
This project is an interactive Sales & Revenue Analysis Dashboard developed using Python, Streamlit, Pandas, and Plotly. It helps analyze sales performance, revenue trends, top-performing products, and regional sales through interactive charts and filters.

## 🚀 Features
- Upload CSV/Excel dataset
- Interactive Region & Category Filters
- KPI Cards (Total Sales, Revenue, Orders)
- Revenue Trend Analysis
- Top Performing Products
- Region-wise Sales Distribution
- Category-wise Revenue Analysis
- Dataset Preview

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- OpenPyXL

## 📂 Project Structure
```
app.py
sales_data.csv
requirements.txt
README.md
```

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Dashboard Output
This dashboard provides interactive visualizations for:
- Sales Analysis
- Revenue Analysis
- Top Products
- Region-wise Sales
- Category-wise Revenue
